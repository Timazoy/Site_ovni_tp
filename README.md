# bootstrap_git

Ce site a pour objectif de présenter un exemple de mes compétences en développement web nottament en Bootstrap et Github, en prenant comme sujet une série que j’ai regardée.

Le site est basé sur la série **OVNI(s)** et se compose de trois pages :

- **Accueil** : présentation de la série, bande-annonce et quelques avis.
- **Personnages / Équipes** : présentation des personnages principaux avec une courte description, ainsi que la liste complète de l’équipe ayant travaillé sur la série.
- **Saisons** : liste de tous les épisodes avec une image et une description pour chacun.

## Installation et utilisation

Pour accéder au site :
1. Téléchargez le projet.
2. Ouvrez le fichier `index.html`.
3. Copiez le chemin d’accès du fichier dans votre navigateur.
